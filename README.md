# go-movie-api

## Movie API with GO

## Entity 
- Movie
- Director
- Actor

## Endpoints
### Movie
- GET /movies
- GET /movies/{id}
- POST /movies

### Director
- GET /directors
- GET /directors/{id}
- POST /directors

### Actor
- GET /actors
- GET /actors/{id}
- POST /actors

## Database
- Postgres

## ORM
- GORM

## Router
- Gin
